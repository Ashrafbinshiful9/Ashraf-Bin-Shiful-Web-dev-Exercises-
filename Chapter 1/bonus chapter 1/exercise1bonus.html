HTML & CSS Quick Reference Guide

This is a beginner-friendly guide for basic HTML tags and CSS properties.

HTML Tags

<h1> to <h6>

Used to define headings.

<h1>Heading 1</h1>
<h2>Heading 2</h2>

<p>

Used to write a paragraph.

<p>This is a paragraph.</p>

<a>

Used to create a link.

<a href="https://example.com">Click here</a>

<img>

Used to show an image.

<img src="image.jpg" alt="Image">

<ul> & <li>

Used to create a list.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<div>

Used to make a section.

<div>This is a box</div>

<span>

Used to style small text inside a sentence.

<p>This is a <span>highlighted</span> word.</p>

<br>

Used to break a line.

This is a line<br>This is a new line

<hr>

Used to add a horizontal line.

<hr>

<form> & <input>

Used to take input from users.

<form>
  <input type="text">
</form>

CSS Properties

color

Changes text color.

p {
  color: blue;
}

background-color

Changes background color.

body {
  background-color: lightgray;
}

font-size

Changes text size.

h1 {
  font-size: 24px;
}

text-align

Aligns text.

h1 {
  text-align: center;
}

border

Adds a border around elements.

img {
  border: 1px solid black;
}

padding

Adds space inside an element.

div {
  padding: 10px;
}

margin

Adds space outside an element.

div {
  margin: 10px;
}

font-family

Changes the font style.

body {
  font-family: Arial;
}
